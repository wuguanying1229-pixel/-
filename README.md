# NEON ARENA - 貪食蛇大作戰

A high-octane multiplayer snake game with neon aesthetics and smooth gameplay.

## 🎮 Features

- **Neon Visual Style**: Cyberpunk-inspired UI with glowing effects
- **Multiplayer Arena**: Compete against 8 AI snakes
- **Boost Mechanic**: Speed up your snake to outmaneuver opponents
- **Leaderboard**: Real-time ranking system
- **Minimap**: Bird's-eye view of the arena
- **Smooth Controls**: Mouse or keyboard-based direction control

## 🚀 Play Online

**[🎯 Play NEON ARENA](https://wuguanying1229-pixel.github.io/-/)**

The game is automatically deployed to GitHub Pages on every push to the main branch.

## 🎮 How to Play

1. Enter your snake name
2. Click **"進入戰局"** (Enter Battle) to start
3. **Control**: Move your mouse or use arrow keys/WASD
4. **Boost**: Left-click or press spacebar to accelerate
5. **Objective**: Grow your snake and eliminate opponents to top the leaderboard

## 📋 Game Mechanics

- **Growth**: Collect particles to grow your snake
- **Boost**: Costs length but provides speed advantage
- **Collision**: Crashing into other snakes ends your game
- **Boundary**: Stay within the arena or you're eliminated
- **Respawn**: AI snakes respawn after elimination; you can try again

## 🛠️ Technical Stack

- Pure HTML5 Canvas
- Vanilla JavaScript (no frameworks)
- CSS3 with CSS Variables for theming
- Automated deployment via GitHub Actions

## 📦 Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions. Any push to the `main` branch triggers an automatic build and deployment.

### Local Development

Simply open `index.html` in your browser to play locally.
